## Operating Rule

These are production packets only. Do not start HeyGen jobs, ElevenLabs voice generation, n8n audio regeneration, social publishing, or queue insertion from this folder without explicit approval.

## Compatibility Notes

- The local video idea route expects a title, spoken script, and storyboard with 3-6 scenes.
- The HeyGen generation route rejects scripts over 5,000 characters.
- Template mode should use the AmaduTown HeyGen template and brand voice when configured.
- Avatar fallback should use the configured HeyGen avatar and voice IDs.
- ElevenLabs remains useful for audio-only social cutdowns or voiceover variants through the social-content audio regeneration path, not for this packet's direct generation step.
