## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

### Projects

Edit `components/Projects.tsx` to add your own projects. Update the `projects` array with:
- Project title and description
- Technologies used
- Project image URL
- GitHub and live demo links

### YouTube Videos

Edit `components/Videos.tsx` to add your YouTube videos. Replace the `videoId` values with your actual YouTube video IDs.

### Social Links

Update the social links in `components/Contact.tsx` with your actual profiles.

### About Section

Modify the content in `components/About.tsx` to reflect your skills and experience.

## Build for Production

```bash
npm run build
npm start
```
