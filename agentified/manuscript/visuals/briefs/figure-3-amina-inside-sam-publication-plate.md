# Figure brief: A.M.I.N.A. inside SAM publication plate

Status: restart brief

## Figure role

This is the flagship Agentified figure.

It should make the reader understand the book's core claim:

```text
Acceleration in an agentic world only scales when the process can be trusted.
```

The figure should show that A.M.I.N.A. is not a separate chatbot sitting beside SAM. A.M.I.N.A. is the trust engine inside SAM. It turns speed into evidence a human can inspect.

## Reference standard

Use the original "Accelerated" SAM illustration as the quality bar:

- `/Users/vambahsillah/Projects/Portfolio/agentified/source-assets/accelerated/accelerated-sam-infographic.png`

Borrow the level of detail, the instructional density, the gold-on-dark editorial feel, and the sense of movement.

Do not copy the original layout one-for-one. This figure should feel like an extension of the concept, not a duplicate.

## Required concepts

SAM foundation:

- Signals: raw noise becomes a usable signal.
- Alignment: insight becomes shared direction.
- Momentum: tested learning becomes a system that compounds.

A.M.I.N.A. governance engine:

- Align: bound the work.
- Map: name authority.
- Instrument: attach receipt.
- Negotiate: set the gate.
- Audit: feed learning back.

Receipt trail:

- Ask: what did the human want?
- Act: what did the agent do?
- Attest: what can be verified?

## Recommended composition

Use a publication plate, not a node diagram.

Suggested structure:

1. A large SAM engine sits in the center, with three visible zones: Signals, Alignment, Momentum.
2. A.M.I.N.A. lives inside the lower half of the SAM engine as a governed mechanism, not as a row outside the circle.
3. Each A.M.I.N.A. step touches or protects a part of SAM:
   - Align protects Signals from vague intent.
   - Map protects Alignment from unnamed authority.
   - Instrument turns action into a receipt.
   - Negotiate keeps consequence behind a gate.
   - Audit feeds verified learning back into Momentum.
4. The receipt trail should run below or through the engine: Ask -> Act -> Attest.
5. A dashed or glowing feedback path should return from Attest into the SAM loop.

## Visual language

Use:

- dark textured navy field
- warm gold linework and headers
- ivory panels for readable labels
- muted teal for verified proof
- restrained red only for unresolved risk, if used at all
- icon-led teaching: source funnel, authority badge, receipt card, human gate, eval loop, proof seal

Avoid:

- plain circles with labels
- acronym pills floating around a shape
- straight connector lines that ignore the SAM curve
- generic flowchart boxes
- AI-generated text baked into the image
- decorative complexity that does not teach the process

A.M.I.N.A. inside SAM

The trust-layer engine inside governed acceleration

Bottom thesis:

A.M.I.N.A. turns SAM into a governed operating loop.

Caption:

```text
A.M.I.N.A. is the operating habit that keeps SAM trustworthy. Align the work, map the authority, instrument the receipt, negotiate the gate, and audit the outcome before expanding permission.
```

## Print requirements

Produce at least:

- editable source file
- high-resolution PNG, minimum 3600 px wide
- PDF export for print review
- SVG or vector export if the tool supports it cleanly
- black-and-white or grayscale review image

The physical book version must be readable at 6x9 trim. If the smallest labels cannot survive print, move those words into the caption or simplify the plate.

## Review question

When reviewing the first proof, ask one question:

```text
Can a reader understand why A.M.I.N.A. belongs inside SAM without reading a paragraph first?
```

If the answer is no, the figure is not ready.
