# Agentified illustration bases

This folder is for textless illustration bases used by the publication-plate workflow.

Rules:

- Keep generated or commissioned art bases textless.
- Do not bake final labels, acronyms, captions, figure numbers, or small copy into the image.
- Preserve the original art file and source note.
- Use `/Users/vambahsillah/Projects/Portfolio/agentified/scripts/composite-amina-sam-publication-plate.mjs` to add exact typography and export review assets.
