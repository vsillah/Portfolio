# Agentified cover bases

This folder is for textless cover base images used by the Agentified cover-comps workflow.

Rules:

- Keep generated or commissioned cover bases textless.
- Do not bake final labels, titles, subtitles, author names, imprint copy, or small cover text into the image.
- Preserve the generated source note in `/Users/vambahsillah/Projects/Portfolio/agentified/manuscript/visuals/source-map.md`.
- Use `/Users/vambahsillah/Projects/Portfolio/agentified/scripts/composite-agentified-cover-variants.mjs` to add exact typography and export review comps.
